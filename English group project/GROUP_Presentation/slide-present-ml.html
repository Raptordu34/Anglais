From Rules to Learning

Machine Learning
Instead of programming rules manually — the machine finds patterns on its own by training on examples.
Example: You don't tell it "a cat has pointy ears."
You show it 10,000 photos of cats — it figures it out.

The Hierarchy

  • AI — the broad field
  • Machine Learning — learning from data
  • Deep Learning — multi-layer neural networks
  • LLMs — large language models
